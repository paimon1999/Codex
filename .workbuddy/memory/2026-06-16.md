## 关键修复：macOS 透明窗口
- **问题1**：`WebviewWindowBuilder::transparent()` 在 macOS 上需要 `macos-private-api` feature
- **修复**：
  1. `Cargo.toml` tauri features 加 `"macos-private-api"`
  2. `tauri.conf.json` 加 `"app": { "macOSPrivateApi": true }`
  3. `lib.rs` 主窗口和 overlay 窗口都加 `.transparent(true)`
- **问题2**：`set_effects(Popover)` 但 body 背景黑色（WebView 默认背景色不透明）
- **修复**：`set_background_color(Some(Color(0,0,0,0)))` 在进入浮动模式时同时设窗口背景透明
  - macOS 上 `set_background_color` 仅对 window 层生效（webview 层未实现）
  - 需配合 CSS `body.floating-mode { background: transparent }` 让 WebView 层也透明
- **问题3**：用户要求白色毛玻璃而非黑色
- **修复**：`body.floating-mode .floating-island` 改为白色半透明 `rgba(255,255,255,0.35)` + 白色 box-shadow
- **清理效果**：`set_effects(None::<WindowEffectsConfig>)` 而非 `EffectsBuilder::new().state(Inactive).build()`
- **capabilities**：新增 `core:window:allow-minimize`、`core:window:allow-set-effects`、`core:window:allow-set-background-color`
