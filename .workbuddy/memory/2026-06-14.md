1. **创建 Vite 插件 `vite-plugin-git-api.ts`**：

   - 提供 REST API 端点：
     - `/api/git/info` — 获取 Git 仓库信息（分支、提交、远程仓库）
     - `/api/git/changes` — 获取工作区变更文件列表
     - `/api/git/diff` — 获取指定文件的 diff
     - `/api/git/branches` — 获取分支列表
     - `/api/git/log` — 获取提交历史
     - `/api/git/status` — 获取仓库状态
     - `/api/git/upstream-diff` — 获取与上游的差异
     - `/api/git/fetch-upstream` — Fetch 上游代码

2. **重构前端 UI**：
   - 动态数据：所有数据从 API 获取，不再硬编码
   - 四个 Tab 页：工作区变更、上游对比、提交历史、分支列表
   - 自动刷新：每 3 秒自动刷新（可暂停/恢复）
   - 实时状态栏：显示 staged/modified/untracked/deleted 计数
   - 文件详情：点击文件查看完整 diff
   - 搜索过滤：支持文件路径搜索

3. **白色主题**：
   - `#ffffff` 背景、`#f8f9fb` 表面、`#e2e4ea` 边框
   - Diff 代码对比：绿色=新增、红色=删除、蓝色=hunk
